Projetos em Destaque

De ideias abstratas a projetos bem-sucedidos, aqui estão alguns dos meus trabalhos mais recentes

  • Criptografia homomórfica

    Este projeto implementa criptografia homomórfica utilizando o algoritmo Paillier para proteger dados em uma aplicação C. Ele criptografa e descriptografa strings usando operações seguras com base em chaves públicas e privadas. O projeto foi desenvolvido em C, utilizando a biblioteca GMP para operações aritméticas de alta precisão e a biblioteca Paillier para implementar a criptografia homomórfica. O ambiente de desenvolvimento escolhido foi o WSL (Windows Subsystem for Linux), que facilitou a compilação e execução do código, permitindo a integração com as bibliotecas criptográficas necessárias e o acesso ao /dev/urandom para geração de números aleatórios.
